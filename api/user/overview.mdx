---
title: "User API"
description: "Endpoints available to field surveyors with the User role."
---

The User role is assigned to front-line field surveyors. Users are scoped to one or more constituencies and are responsible for conducting door-to-door surveys, registering unregistered voters, and recording voter card scans on election day.

<Note>All endpoints under `/api/user/*` require the **User** role and a valid JWT bearer token.</Note>

<Note>Page-level access (which sections of the app are visible) is controlled by the Admin through the pages permission system. Use `GET /api/user/fatch-all-user-permissions` to retrieve the current user's page access flags at login.</Note>

## Voter lists

|--------|------|-------------|
| `GET` | `/api/user/voters-list` | Paginated list of voters in assigned constituencies |
| `GET` | `/api/user/voters-list/newly-registered` | Newly registered voters |

| `GET` | `/api/user/birthday-voters-contacted/{id}` | Birthday voters who have been contacted |
| `GET` | `/api/user/voters-diff-address` | Voters registered at a different address |
| `GET` | `/api/user/voters-history/{id}` | Survey history for a specific voter |
| `GET` | `/api/user/voters/address` | Address-based voter search |
| `GET` | `/api/user/voters/unregister/address` | Address search for unregistered voters |
| `GET` | `/api/user/national-registery-list` | National registry voter list |

| `GET` | `/api/user/get-user-surveys-count` | Total survey count for the current user |

## Unregistered voters

Unregistered voters are individuals who have been identified in the field but do not appear in the official voter register. The contact-tracking workflow uses notes to record each outreach attempt.

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/user/unregistered-voters` | Add a new unregistered voter |
| `GET` | `/api/user/unregistered-voters` | List unregistered voters in scope |
| `GET` | `/api/user/unregistered-voters/{id}` | Get a single unregistered voter |
| `PUT` | `/api/user/unregistered-voters/{id}` | Update an unregistered voter |
| `DELETE` | `/api/user/unregistered-voters/{id}` | Delete an unregistered voter |
| `GET` | `/api/user/unregistered-voters-diff-address` | Unregistered voters at different addresses |
| `POST` | `/api/user/update-diff-address-unregister-voter` | Update address flag for an unregistered voter |
| `GET` | `/api/user/unregistered-voters/{id}/get-contacted_voters` | Records of contacted attempts |
| `GET` | `/api/user/unregistered-voters/{id}/get-uncontacted_voters` | Records of uncontacted attempts |

### Notes (contact tracking)

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/user/unregistered-voters/notes/list/{unregistered_voter_id}` | List notes for an unregistered voter |

## Voter cards

On election day, field users scan or record voter cards to track turnout in real time.

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/user/add-voter-card-result` | Record a voter card scan |
| `GET` | `/api/user/get-voter-card-result/{id}` | Get a voter card result |
| `POST` | `/api/user/update-voter-card-result/{id}` | Update a voter card result |
| `DELETE` | `/api/user/delete-voter-card-result/{id}` | Delete a voter card result |
| `GET` | `/api/user/list-voter-card-result` | List all voter card results for this user |
| `GET` | `/api/user/get-voter-with-id/{id}` | Look up voter details by ID |
| `POST` | `/api/user/upload-voter-card` | Upload a voter card image |
| `GET` | `/api/user/get-voter-card-images` | Retrieve uploaded voter card images |
