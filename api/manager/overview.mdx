---
title: "Manager API"
description: "Endpoints available to users with the Manager role."
---

The Manager role sits between Admin and field User. A manager oversees one or more constituencies, supervises field users working in those areas, and has read/write access to surveys, voter cards, and reports scoped to their assigned constituencies.

<Note>All endpoints under `/api/manager/*` require the **Manager** role and a valid JWT bearer token.</Note>

<Tip>A manager with `is_coordinator = true` has visibility across multiple constituencies simultaneously.</Tip>

## Voter cards

Voter cards track whether a canvasser has physically collected or viewed a voter's electoral card, broken out by party affiliation.

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/manager/voter-cards-fnm` | FNM voter cards |
| `GET` | `/api/manager/voter-cards-fnm/export` | Export FNM voter cards as PDF |
| `GET` | `/api/manager/voter-cards-plp` | PLP voter cards |
| `GET` | `/api/manager/voter-cards-dna` | DNA voter cards |
| `GET` | `/api/manager/voter-cards-unk` | Unknown-party voter cards |
| `POST` | `/api/manager/add-voter-card-result` | Record a voter card scan result |
| `GET` | `/api/manager/get-voter-card-result/{id}` | Get a voter card result |
| `POST` | `/api/manager/update-voter-card-result/{id}` | Update a voter card result |
| `DELETE` | `/api/manager/delete-voter-card-result/{id}` | Delete a voter card result |
| `GET` | `/api/manager/list-voter-card-result` | List all voter card results |
| `GET` | `/api/manager/get-voter-with-id/{id}` | Get voter details by ID |

## Users management

Managers can create and manage field users (surveyors) within their assigned constituencies.

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/manager/users` | List field users in scope |
| `POST` | `/api/manager/users` | Create a field user |
| `GET` | `/api/manager/users/{user}` | Get a single field user |
| `PUT` | `/api/manager/users/{user}` | Update a field user |
| `DELETE` | `/api/manager/users/{user}` | Delete a field user |
| `GET` | `/api/manager/users/surveys/{id}` | Get surveys for a specific user |
| `GET` | `/api/manager/get-user-surveys-count/{id}` | Get survey count for a specific user |
| `GET` | `/api/manager/search-constituencies` | Constituency search for user forms |
| `GET` | `/api/manager/get_constituencies` | Full list of constituencies in scope |
